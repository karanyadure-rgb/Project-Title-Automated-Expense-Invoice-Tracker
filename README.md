# Automated Expense & Invoice Tracker

A Python application that reads employee expense records, converts amounts
to a base currency using live exchange rates, flags expenses that exceed
daily limits, and generates a formatted Excel/CSV summary report.

Built as a hands-on project to practice Python fundamentals through
external libraries — from basic loops to OOP, file I/O, error handling,
and working with real APIs.

## Features
- Reads raw expense data from CSV
- Converts multi-currency expenses to a base currency (INR/USD) using live rates
- Flags expenses that exceed company daily limits
- Groups and summarizes expenses by employee
- Outputs a clean, formatted Excel report

## Tech stack
Python, requests, openpyxl (or pandas)
